## Overview

This app  is a **social media app prototype** developed using **Flutter**.  

The app is a functional recreation of a social media platform, designed for educational purposes in the **Mobile Programming ** subject.  

It includes **login functionality, a user profile, newsfeed, notifications, and ads**, but it is **offline-only and does not use a database**.  

The project focuses on **UI design, navigation, and app structure**, giving students practical experience in building Flutter apps with typical social media features.

---

## Features

- **Login Screen** – User authentication simulation  
- **Profile Page** – Displays user information and editable placeholders  
- **Newsfeed** – Scrollable posts with images and text  
- **Notifications** – Displays simulated alerts and updates  
- **Advertisements** – Static or placeholder ads for demonstration  
- **Offline Functionality** – No database required, all data is in-app  
- **Flutter Widgets** – Utilized common widgets like `ListView`, `AppBar`, `Drawer`, `Cards`, etc.
